1- Comando inicial para verificar as informações no BD:

DICA 1 - todos os comandos Mysql devem terminar com ponto e vírgula.
DICA 2 - a setinha para cima do teclado repete os últimos comandos digitados.

Para listar oa BD de dados existentes devemos utilizar o comando: show databases;

- o Mysql cria em sua instalação alguns Bancos de Dados para teste, como exemplo o test.

2 - Criando e acessando seu próprio BD:

- Para criar um BD devemos utilizar o comando create database

Ex: create database aula;

- Sempre que criamos um BD devemos acessa-lo, para acessarmos um BD devemos utilizar o comando: use

Ex: use aula;

- Observe atentamente a resposta a cada comando: Query Ok - siginifica que o comando foi realziado com sucesso.
- No caso do acesso ao BD, o retorno foi Database Changed.
- Digite novamente o comando Show Databases para verificar se o seu BD foi criado corretamente.

- a partir de agora já podemos começar a criar as tabelas e manipular nossas informações:

3 - Criando nossa tabela base

- Neste curso iremos utilizar apenas uma tabela, que terá o nome de funcionario.
- Lembrando que os nomes de tabelas e campos não podem conter acentuação, espaços ou caracteres especiais.

Modelo da Tabela:

- Para criarmos a tabela acima devemos digitar o comando abaixo no Terminal Mysql, confira corretamente as vírgulas e parênteses.
- Digite tudo na sequência, não precisa pular linha, a quebra é automática.
- Caso você feche o terminal, nada é perdido, você precisa somente acessar o BD novamente através do comando use.

Comando:
CREATE TABLE funcionarios (id int key auto_increment, nome varchar(60), email varchar(60), salario float(10,2), data_nascimento date, cidade varchar(60), estado varchar(2));

3 - Listando e Visualizado a Estrutura da tabela:

- Para listar as tabelas existentes no BD devemos utilizar o comando: show tables;
- Para verificar os campos e a estrutura da tabela, devemos utilizar o comando Describe ou somente DESC: desc funcionarios;

- Temos que ter muita atenção ao nome do campo e ao tipo de cada campo, pois será essencial para nosso aprendizado.
