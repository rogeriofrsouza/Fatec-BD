1 - Inserindo registros na Tabela:

Dicas Importantes:

- Para inserir um registro corretamente é preciso obedecer a ordem dos campos na tabela. Use o comando DESC para consultar a ordem.
- Todos os campos do tipo Varchar, Text e Date precisam de aspas.
- Os campos do tipo Numeric, Integer, Decimal e Float não precisam de aspas.
- No Terminal o atalho CTRL+V não funciona, é preciso clicar com o botão direito do mouse e escolher a opção Colar.

 

- Para inseir um registro na tabela devemos utilizar o comando INSERT

Exemplo:

INSERT INTO funcionarios VALUES (Null, 'Luis Paulo', 'luispaulosj@gmail.com', 2000.00,'1980-07-10','São José do Rio Pardo','SP');

Observações:
- Utilizamos o Null pois o campo id é auto_increment
- O separador de decimais no BD é o ponto
- A data deve ser no formato Ano-Mês-Dia
- Se atente a dica inicial sobre as Aspas
- Novamente não pressione ENTER enquanto digita, digite tudo em sequência.

 

- Para a inserção de mais registros na tabela é só seguir o modelo acima e trocar as informações:


Exemplo:


INSERT INTO funcionarios VALUES (Null, 'João', 'joao1970@gmail.com', 8000.00,'1970-05-20','São Paulo','SP');

- Atenção ao retorno do Comando: Query Ok.
