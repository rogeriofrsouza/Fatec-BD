1 - Comando UPDATE

- Para alterar um registro e uma tabela usamos o comando UPDATE, seguindo a sintaxe abaixo:

UPDATE tabela
SET coluna = novo_valor_armazenado
WHERE coluna = valor_filtro;

Observação importante: Caso não seja usada a cláusula WHERE para filtrar os registros, todos os dados da coluna serão alterados!

Exemplo: Vamos alterar o salário do funcionário de id=5 para 6000.

UPDATE funcionarios SET salario=6000 where id=5;

- observe no exemplo acima, que primeiro foi realizado um select para verificar o salário atual, depois do comando update realizado executei novamente o select para verificar se o salário foi devidamente alterado.

- Nunca se esqueça de utilizar a clausula WHERE junto ao update, senão todos os dados da coluna da tabela serão alterados.
- No exemplo acima, caso eu não especificasse ID=5, todos os funcionários passariam a ter salário de 6000.
- Procure utilizar na comparação da clausula WHERE sempre o ID ou algum campo Chave Primária.

 

Exemplo 2: Vamos alterar agora o salário e o email do mesmo funcionario. Observem que quando eu altero mais de uma coluna, eu devo separa-las com vírgula.

UPDATE funcionarios SET salario=4000, email='fer@gmail.com' where id=5;

- Observe que sempre que utilizamos o Update é necessário o uso do Select para verificarmos as informações.
- Atenção novamente para o uso das ASPAS em campos do tipo varchar.

 

2 - Comando DELETE e TRUNCATE

- Comandos utilizados para remover registros de uma tabela.

- Para remover um determinado registro da tabela, utilize o comando DELETE:

Exemplo:

DELETE FROM funcionarios where id=5;

- Observe no exemplo acima que informamos o ID=5, e no segundo select o mesmo não aparece mais.

3 - Truncate

- Apaga todos os registros da tabela.

Exemplo:

TRUNCATE funcionarios;

- Observe na imagem acima, que após o comando TRUNCATE, nenhum registro foi exibido no select.
- Embora consigamos o mesmo resultado com o DELETE, é indicado neste casos a utilização do Truncate.
